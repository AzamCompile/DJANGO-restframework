📌 Django REST Framework CRUD API
📖 About the Project

This is a simple CRUD (Create, Read, Update, Delete) REST API built with Django REST Framework.
The project demonstrates how to manage data using RESTful endpoints with Django models and serializers.

⚙️ Features
Create new records
Retrieve single or all records
Update existing records
Delete records
RESTful API structure
Django ORM integration
🛠️ Tech Stack
Python 3
Django
Django REST Framework
SQLite / PostgreSQL
